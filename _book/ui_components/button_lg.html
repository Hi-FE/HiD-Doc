Introduction

button-lg

buttom-lg是一种尺寸较大的按钮，它的内容由一个icon、一个标题和一个描述组成。

  <button-sm name="普通" desc="新增一个普通类型的聚合页" icon="i-plus"></button-sm>
  • name string 按钮标题，必须。
  • desc string 按钮功能描述，必须。
  • icon string 按钮示意图标的名称，必须。

暂时只有绿色的样式。
默认设置inline-block，若需要指定宽度请注意设置为block。
